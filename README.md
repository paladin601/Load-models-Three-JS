# Controles3JS
Controles para VR threeJS, in development 

Usar el siguiente link para instalar el webpack
 ```
 https://webpack.js.org/guides/getting-started/ 
 ```
 
 Para realizar cambios buscar el archivo  `index.js` 
 
 # Dependencies
 
  - Webpack
  - ThreeJS
